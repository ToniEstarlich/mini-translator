# Mini English → Japanese Translator

## Description
A mini web application built with **Flask** to translate English words into Japanese, show their reading in **Kana and Romaji**, and fetch related images using external APIs.  
This project is designed to **practice integrating and handling APIs** in Python and Flask.

## Technologies
- Python, Flask  
- Jinja2 (templates)  
- JavaScript (optional)  
- APIs: MyMemory (translation) and Pexels (images)  
- `pykakasi` for Kana and Romaji conversion  

## Features
- English → Japanese translation  
- Display reading in Kana and Romaji  
- Fetch an image related to the word  
- Error handling and fallback if API fails
